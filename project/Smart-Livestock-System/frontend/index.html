About This Project

  • Web Technologies: HTML5, CSS3, JavaScript, PHP, MySQL
  • Algorithms: Greedy, Binary Search, Dynamic Programming, Graph Traversal
  • OS Concepts: Process Management, CPU Scheduling, File & Memory Management

Built with production-ready standards including MVC architecture, error handling, and responsive design.
